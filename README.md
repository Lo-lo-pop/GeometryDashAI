В данном проекте находиться ИИ по гд у которого есть такие функции как:
>Компютерное зрение и обнаружение перед игроком блоков (работает так себе)\
>Обучение и сохранение в память ИИ\
>Играть на обученой ИИ модели в разные уровни

**Как запустить?**\
Для начала скачайте все нужные библиотеки через:
pip install -r requirements.txt

После запускайте файл: 
main.py\
И дальше будет меню консоли нажимаете 1 и ИИ начнёт учиться играть
